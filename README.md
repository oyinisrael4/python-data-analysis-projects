# Python Data Analysis Projects

## What This Project Is About
This notebook contains different data analysis projects 
I worked on while learning Python. Each one used a 
different real world dataset and taught me something 
new about working with data.

## Projects Inside

### NYC Weather Analysis
I loaded a CSV file of New York weather data from January 
2016 and cleaned it up. Some columns had missing values 
that I filled in and one column had a special character 
"T" for trace rainfall that I had to handle separately. 
I then made 6 interactive charts using Plotly to explore 
the data visually.

### Global Earthquake Analysis
This one used over 23,000 earthquake records going back 
to 1969. I connected to MySQL to get the data, then 
analysed things like which locations had the most 
earthquakes and how magnitude related to depth. I also 
plotted the earthquakes on an interactive world map.

### UK Energy Consumption Analysis
This was the largest dataset I worked with, over 155,000 
records of UK power consumption from 2015 to 2023. I 
extracted time features like hour, day and month from 
the datetime column and found clear seasonal patterns. 
For example power consumption is highest in winter 
evenings and lowest in summer mornings.

### Python Maths and Problem Solving
I used Python to solve geometry problems like calculating 
the volume of cones and spheres, and also wrote code to 
solve quadratic equations using the quadratic formula.

## What I Learned
Each dataset came with its own challenges. The energy 
dataset taught me how to work with very large files 
efficiently. The weather data taught me how to handle 
unusual values that are not technically null but are 
still wrong. Overall these projects helped me get 
comfortable moving from raw data to actual insights.

## Technologies Used
Python, pandas, numpy, matplotlib, 
Plotly, seaborn, MySQL, Jupyter Notebook
